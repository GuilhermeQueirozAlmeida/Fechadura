# Fechadura
<div>Este projeto t&ecirc;m como objetivo, mostrar um uso pr&aacute;tico do Arduino.</div><div>&nbsp;</div><div>Um sistema de seguran&ccedil;a para fazer o controle de acesso, por exemplo, da entrada de pessoas a um determinado ambiente seguro.</div><div>&nbsp;</div><div>O sistema consta de um teclado para a entrada de uma senha, e um display LCD para visualizar mensagens.</div><div>&nbsp;</div><div>A simula&ccedil;&atilde;o &eacute; feita com LEDs, por&eacute;m facilmente implementado para rel&eacute;s ou solen&oacute;ides.</div><div>&nbsp;</div><div>Como funciona:</div><div>&nbsp;</div><div>Ao iniciar o sistema, um LED vermelho indica que o sistema est&aacute; pronto, e neste caso n&atilde;o adianta digitar n&uacute;meros no teclado, pois os mesmos n&atilde;o ser&aacute; processados; a n&atilde;o ser a tecla "#", que indicar&aacute; ao sistema que algu&eacute;m ir&aacute; digitar uma senha.</div><div>Ao digitar "#"o sistema libera o teclado para a entrada da senha, e um LED amarelo sinaliza esta funcionalidade, bem como, ser&aacute; mostrado uma mensagem no display informar ao usu&aacute;rio que &eacute; necess&aacute;rio digitar a senha.</div><div>Digitando n&uacute;meros no teclado (para cada um ser&aacute; emitido uma sinaliza&ccedil;&atilde;o ac&uacute;stica), o sistema processar&aacute; os dados. &nbsp;</div><div>Se a quantidade de n&uacute;meros digitar for igual a 4 e corresponder exatamente a senha do sistema, ent&atilde;o ser&aacute; habilitato o rel&eacute; ou o solen&oacute;ide de acionamento (indicado por um LED verde, que ir&aacute; piscar por cerca de 5 segundos, e o sistema estar&aacute; liberado; durante esse per&iacute;odo um som ser&aacute; emitido para sinaliza&ccedil;&atilde;o; uma mensagem ser&aacute; mostrada no display, para informar ao usu&aacute;rio que o acesso foi liberado).</div><div>Se a quantidade de n&uacute;meros digitados for 4 e n&atilde;o corresponder a senha esperado para o sistema, ent&atilde;o o sistema n&atilde;o ser&aacute; liberado e voltar&aacute; ao estado inicial. &nbsp;O mesmo acontece se for pressionado a tecla "#" novamente.</div><div>&nbsp;</div><div><strong>O HARDWARE:</strong></div><div>1 x Arduino UNO</div><div>1 x Teclado num&eacute;rico 3x4</div><div>1 x Display LCD 2x16 I2C</div><div>1 x Buzzer</div><div>1 x Breadboard</div><div>4 x Resistores de 220O 1/8W</div><div>1 x LED verde 3mm</div><div>1 x LED Amarelo 3mm</div><div>1 x LED Vermelho 3mm</div><div>Fios e cabos para conex&otilde;es</div><div>&nbsp;</div><div><strong>O CIRCUITO:</strong></div><div>Utilize a breadboard para fazer as liga&ccedil;&otilde;es:</div><div>&nbsp;</div><table class="tr-caption-container" cellspacing="0" cellpadding="0" align="center"><tbody><tr><td><img src="https://2.bp.blogspot.com/-0-9lrZL_CyI/T1oaYNAGb2I/AAAAAAAAAH4/xSH7qCrWfQc/s200/breadboard+mini.jpg" width="200" height="150" border="0" /></td></tr><tr><td class="tr-caption">Mini BreadBoard</td></tr></tbody></table><table class="tr-caption-container" cellspacing="0" cellpadding="0" align="center"><tbody><tr><td><a href="https://4.bp.blogspot.com/-9ZoJmTsdjo0/T1oaMZ7-dBI/AAAAAAAAAHw/Or9AjTliLd4/s1600/Breadboard+1.jpg"><img src="https://4.bp.blogspot.com/-9ZoJmTsdjo0/T1oaMZ7-dBI/AAAAAAAAAHw/Or9AjTliLd4/s200/Breadboard+1.jpg" width="200" height="200" border="0" /></a></td></tr><tr><td class="tr-caption">BreadBoard</td></tr></tbody></table><div class="separator">&nbsp;</div><div>&nbsp;</div><div>&nbsp;</div><div>Para cada LED associar um resistor de 220O&nbsp;em s&eacute;rie conectado ao terminal Anodo e o terminal catodo ligado diretamente ao negativo da alimenta&ccedil;&atilde;o.
